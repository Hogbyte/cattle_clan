Project Name: The Evil Epsilon Base

DESCRIPTION:

	The Evil Epsilon base will be built on the far west end
of the grove.  This will give us more options of where to start
game play from. This will bring the action to different areas of
the grove.

What is an Evil Epsilon?

	The epsilon is a Greek letter that looks like E.  We used
epsilon in my Calculus class to represent error.  The perfect error
would be zero.  However, because epsilon was in the denominator it
could not be zero.  So we had to use algebraic manipulation to find
a "good epsilon" in the numerator to cancel out the "evil epsilon"
in the denominator.
